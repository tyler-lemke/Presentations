# Robot Framework
## https://robotframework.org/

# Robot Framework Characteristics
- Used for Acceptance Testing and RPA
- Uses BDD syntax for tests
- Has a ton of libraries

# Katalon Recorder
## https://www.katalon.com/resources-center/blog/katalon-automation-recorder/

# Katalon Recorder Characteristics
 - Chrome/Firefox Extension
 - Hard to run in CI/CD pipeline
 - Limited Functionality - APIs aren't powerful

# Cognizant Intelligent Test Scripter
## https://cognizantqahub.github.io/Cognizant-Intelligent-Test-Scripter-Helpdoc/index.html
# Cognizant Intelligent Test Scripter Characteristics
- Web and App Testing
- Accessibility Testing
- Layout Testing
- Selenium Grid Support

# SikuliX
## http://sikulix.com

# SikuliX Characteristics
- What You See Is What You Script 100% Visual
- Can be used for RPA
- Automates other tasks you don't have the tools to automate

# cypress-cucumber-preprocessor
## https://github.com/TheBrainFamily/cypress-cucumber-preprocessor
# cypress-cucumber-preprocessor Characteristics
 - Use Gherkin syntax in Cypress
 - Yes, Its really cool

# Code Solutions
# Cypress
## https://cypress.io
# Cypress Characteristics
 - Amazing Feedback and Timeline
 - Simple test setup - write first test in 5 min
 - My Favorite Tool
 - Only runs against Chrome

# Test Cafe
## https://www.devexpress.com/products/testcafestudio/
## Test Cafe Characteristics
 - Alternative to Cypress
 - Runs against Every browser
 - Has a paid component for QAs 

# Appium
## http://appium.io/docs/en/about-appium/getting-started/
# Appium Characteristics
 - Mobile testing and Windows App Testing
 - Uses Selenium
